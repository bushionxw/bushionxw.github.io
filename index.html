1.添加用户

1.1 功能

添加新用户

useradd

1.2 参数含义

-c 添加备注

-d 为主目录制定名字

-e 用YYYYY-MM-DD 指定账户过期日期

-f 指定密码过期后多少天禁用账户，0：一过期就禁用，-1：禁用这个功能

-g 指定用户登陆组的GID或者组名,修改替换默认用户组

-G 指定用户除登陆组之外的所有组，增加新的用户组

-k 和-m一起使用，将/etc/skel目录聂荣复制到HOME目录

-m 创建用户的HOME目录

-M 不创建用户的HOME目录

-n 创建一个与用户登陆同名的新组

-r 创建系统账户

-p 为用户账户指定默认密码

-s 指定默认登陆shell

-u 指定账户的UID

1.3 例子

查看用户默认值

useradd -D

添加新用户:创建Home目录

useradd -m test

2.删除用户

userdel

只删除用户信息（/etc/passwd）：

userdel test

删除用户所有文件:home目录及邮件目录

userdel -r test

3.修改用户

3.1 usermod

3.1.2 参数

-l 修改账户的登录名

-L 锁定账户，

-p 修改账户密码

-U 解除锁定

3.1.3 例子

3.2 passwd

3.2.1 功能

修改用户密码

3.2.2 例子

修改单个用户密码

passwd test

批量修改密码

chpasswd < users.txt

​ users.txt格式：userid:passwd

3.3 修改账户有效期

chage

参数

-d 设置上次修改密码到现在的天数

-E 设置密码过期的日期

-I 设置密码过期到锁定账户的天数

-m 设置修改密码之间最少要多少天

-W 设置密码过期前多久开始提醒

4.用户组

4.1 创建新组

groupadd

例子：

groupadd shared

用户增加用户组(-G:增加，-g:替换修改)

usermod -G shared test

4.2 修改组

groupmod

groupmod -n sharing shared
