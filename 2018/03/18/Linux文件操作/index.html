Linux文件操作

1.修改权限

chmod

1.1 参数

r:4

w:2

x:1

u: 用户

g: 组

o: 其它

a: 所有

1.2 例子

所有的人有rwx权限

chmod 777 test.txt

读权限

chmod u+r test.txt

chmod u-r test.txt

chmod +x test.txt

2. 改变所属关系

2.1 chown

改变属主

chown dan newfile

同时改变属主和属组

chown dan.shared newfile

只改属组

chown .rich newfile

2.2 chgrp

chgrp shared newfile

3. 共享文件

mkdir testdir

chgrp shared testdir

chmod g+s testdir

umask 002

4.文件操作

4.1创建文件

touch test

4.2复制文件

cp test test_one

整个目录复制

cp -R Scripts/ Mod_Scripts

复制多个文件

cp *script Mod_Scripts/

4.3重命名文件

mv

mv fall fzll

mv fzll Pictures/

4.4删除文件

rm

rm -i fall

rm -i f?ll

4.5查看文件

4.5.1查看文件类型

file my_file

4.5.2查看整个文件

cat test1

加上行号查看

cat -n test1

只给非空行加行号

cat -b test1

4.5.3more

显示一页内容：

more test1

4.5.4less

显示一页，可上下页翻动

less test1

4.6查看部分文件

4.6.1tail

显示文件尾部

tail test1

tail -n 2 test1

其他进程可操作文件，刷新内容实时显示

tail -f test1

4.9.2head

显示文件头部

head test1

head -5 test1

5.目录操作

5.1创建目录

mkdir

mkdir new_dir

5.2删除目录

rmdir:只能删除空目录

rmdir new_dir

删除目录及其所有文件：

rm -rf Small_Dir

删除之前可以使用tree工具查看目录文件
