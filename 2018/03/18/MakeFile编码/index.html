MakeFile编码

例子：

#main: test.cpp

#gcc -o main test.cpp

cc = gcc
prom = main
deps = func.h
source = test.cpp
obj = test.o func.o

#deps = $(shell find ./ -name “*.h”)

#src = $(shell find ./ -name “*.c”)

$(prom): $(obj)
$(cc) -o $(prom) $(obj)

%.o: %.c $(deps)
$(cc) -c $< -o $@

clean:
rm -rf $(obj) $(prom)

1.命令

1.1让命令出错了也不停止

​ 在命令前加“-”

​ 如： -rm -f *.o

1.2自动寻找头文件

方法1：编译器增加 -MM选项

​ gcc -MM main.c

​ 方法2：生成.d文件

​ %.d: %.c

​ @set -e; rm -f $@;\

​ $(gcc) -M $(CPPFLAGS) $< > $@.$$$$;\

​ sed ‘s,($).o[:],\1.o $@: ,g’ < $@.$$$$$ > $@; \

​ rm -f $@.$$$$$

1.3引入其它的makefile

​ include

​ 例如：

​ include $(source:.c=.d)

1.4传递变量

​ export

​ 例如：

​ export var = value
